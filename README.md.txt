# Military Asset Management System

A full-stack role-based inventory management platform designed to streamline the allocation, tracking, and expenditure of military assets across bases.

##  Tech Stack

| Layer     | Technology       |
|-----------|------------------|
| Frontend  | React (with JSX, props, routing) |
| Backend   | Spring Boot, Java 17, Spring Security |
| Database  | MySQL (JPA/Hibernate) |
| Auth      | JWT-based Authentication & Role-based Authorization |
| Deployment | Frontend: Netlify · Backend: Render |

## ✨ Features

- 👤 JWT-secured login with roles: `ADMIN`, `COMMANDER`, `LOGISTICS`
- 🧾 Asset management per base (add/view assets)
- 📦 Purchase and assignment modules
- ♻️ Expenditure tracking with filters
- 📍 Role-aware dashboards and navigation
- 🎛️ Configurable access control using Spring Security
